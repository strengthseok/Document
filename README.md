## 함수 정리

- String : 문자열
    - startWith(str) : 해당 문자열로 시작하는지 확인( <-> endWith() )
- Arrays : 배열 관련
    - toString() : 배열 값 보기
    - sort(arr) : 배열값 정렬
    - equals(arr1, arr2) : 배열 비교. ex)Arrays.equals(arr1, arr2) return true/false
- HashMap : 맵 관련
    - getOrDefault(가져올값, 기본값) : 맵에서 값을 가져올때 기본값을 줌
    - values() : 맵의 값들을 가져옴
- Iterator : Collection 요소 순회
    - hasNext() : 다음 값이 있는지 찾음
    - Next() : 다음 값
    - remove() : 요소 삭제
